WordPress plugin to remove authors from the Yoast SEO Sitemap

== Description ==

This WordPress Plugin removes authors from the Yoast SEO Sitemap.

* No setup required!

== Installation ==

Upload the "wp-remove-authors-sitemap-from-yoast-seo" folder to your wp-content/plugins directory, then Activate it.
